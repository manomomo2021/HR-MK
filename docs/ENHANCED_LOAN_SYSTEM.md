# نظام السلف المحسن مع الخصم التلقائي

## نظرة عامة

تم تطوير نظام السلف ليصبح ذكياً ومتقدماً مع خصم تلقائي من الرواتب وواجهة تفاعلية تساعد المستخدمين في اتخاذ القرارات المناسبة.

## 🎯 **الميزات الجديدة**

### **1. جلب الراتب التلقائي عند اختيار الموظف**
```javascript
// عند اختيار موظف، يتم جلب راتبه تلقائياً
const selectedEmployee = employees.find(emp => emp.id === selectedEmployeeId)
const salary = selectedEmployee.salary || selectedEmployee.basicSalary || 0
const maxDeduction = salary * 0.25  // 25% حد أقصى

// إشعار فوري بمعلومات الموظف
toast.success(`
  تم اختيار: ${selectedEmployee.name}
  الراتب: ${formatCurrency(salary, 'EGP')}
  الحد الأقصى للخصم (25%): ${formatCurrency(maxDeduction, 'EGP')}
`)
```

### **2. واجهة تفاعلية محسنة**
- 👤 **معلومات الموظف المالية** تظهر فوراً
- 📊 **حاسبة تفاعلية** للمبالغ المقترحة
- ⚡ **أزرار التعبئة السريعة** للمبالغ الشائعة
- 🚨 **تحذيرات ذكية** مع اقتراحات للحل

### **3. الخصم التلقائي من الرواتب**
- ✅ **خصم شهري تلقائي** عند حساب الرواتب
- ✅ **تحديث حالة السلف** فورياً
- ✅ **إشعارات شاملة** بتفاصيل الخصم
- ✅ **حماية من الخصم الزائد**

## 💰 **نظام الخصم التلقائي**

### **عند حساب الرواتب:**
```javascript
// البحث عن السلف المفعلة للخصم التلقائي
const employeeLoans = loans.filter(l => 
  l.employeeId === employee.id && 
  l.status === 'approved' && 
  l.autoDeduction === true &&
  (l.paidAmount || 0) < l.amount
)

// خصم تلقائي لكل سلفة
for (const loan of employeeLoans) {
  const monthlyDeduction = loan.monthlyDeduction
  const remainingAmount = loan.amount - (loan.paidAmount || 0)
  const actualDeduction = Math.min(monthlyDeduction, remainingAmount)
  
  // تحديث السلفة
  loan.paidAmount += actualDeduction
  if (loan.paidAmount >= loan.amount) {
    loan.status = 'completed'
  }
}
```

## 🔧 **الضوابط والحدود**

### **الحدود القانونية:**
```javascript
const LOAN_LIMITS = {
  maxDeductionPercentage: 25,    // 25% حد أقصى من الراتب
  minRemainingPercentage: 75,    // 75% يجب أن يبقى للموظف
  maxInstallments: 24,           // 24 شهر حد أقصى
  minInstallmentAmount: 100,     // 100 ج.م حد أدنى للقسط
  maxLoanMultiplier: 12          // 12 ضعف الراتب الشهري
}
```

### **شروط الموافقة:**
- ✅ **الموظف نشط** في النظام
- ✅ **الراتب محدد** وأكبر من الحد الأدنى
- ✅ **القسط ضمن الحدود** المسموحة (25%)
- ✅ **عدد الأقساط معقول** (حتى 24 شهر)
- ✅ **لا توجد سلف متعثرة** للموظف
