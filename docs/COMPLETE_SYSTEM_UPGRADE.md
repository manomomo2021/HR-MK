- **رموز العملة** واضحة في كل مكان

## 🎯 **شاشة تعديل الرواتب الاحترافية**

### **التصميم بالتبويبات:**
- 📋 **البيانات الأساسية** - معلومات الموظف
- 💰 **الراتب والبدلات** - الراتب الأساسي والبدلات
- 📅 **الحضور والغياب** - أيام العمل ونسبة الحضور
- ➖ **الخصومات** - جميع أنواع الخصومات
- 🧮 **الملخص** - الحسابات النهائية

### **الحسابات المتقدمة:**
```javascript
// حساب الراتب حسب أيام العمل
const dailySalary = basicSalary / 22  // الراتب اليومي
const actualWorkingDays = 22 - absentDays  // أيام العمل الفعلية
const proportionalSalary = dailySalary * actualWorkingDays  // الراتب النسبي

// الخصومات التلقائية
const absentDeduction = dailySalary * absentDays  // خصم الغياب
const lateDeduction = lateDays * (dailySalary * 0.1)  // خصم التأخير
const insuranceDeduction = grossSalary * 0.09  // التأمينات الاجتماعية
const loanDeduction = loanAmount  // خصم السلف

// الراتب الصافي
const netSalary = grossSalary - totalDeductions
```

### **مثال عملي:**

```
📊 الراتب اليومي: 272.73 ج.م
📊 الراتب النسبي: 5,454.55 ج.م (20/22 يوم)

➖ خصم التأخير: 27.27 ج.م
➖ خصم السلف: 500.00 ج.م
➖ التأمينات: 490.91 ج.م
✅ الراتب الصافي: 3,890.92 ج.م
📈 نسبة الحضور: 90.9%
```

- **تصميم متجاوب** لجميع الشاشات

## 🚀 **الفوائد العملية**

3. **خصم السلف** تلقائياً من الرواتب
4. **تقارير شاملة** للمراجعة

1. **إدارة شاملة** للموظفين والسلف
2. **تتبع دقيق** للحضور والغياب
3. **إحصائيات مفيدة** للقرارات
4. **تصدير سهل** للبيانات
