# شاشة تعديل الرواتب الاحترافية

## نظرة عامة

تم تطوير شاشة تعديل الرواتب بالكامل لتكون احترافية ومتقدمة مع دعم الجنيه المصري وحساب الرواتب حسب أيام العمل الفعلية.

## 🎯 **الميزات الرئيسية**

### **1. تصميم احترافي بالتبويبات**
- **5 تبويبات منظمة** لسهولة التنقل
- **تصميم متجاوب** لجميع الشاشات
- **أيقونات واضحة** لكل قسم
- **ألوان متناسقة** مع النظام

### **2. حساب الرواتب حسب أيام العمل**
```javascript
// حساب الراتب النسبي
const dailySalary = basicSalary / workingDaysInMonth
const actualWorkingDays = workingDaysInMonth - absentDays
const proportionalSalary = dailySalary * actualWorkingDays
```

### **3. خصم السلف والخصومات التلقائية**
- **خصم الغياب**: راتب يومي × أيام الغياب
- **خصم التأخير**: 10% من الراتب اليومي × أيام التأخير
- **التأمينات الاجتماعية**: 9% من الراتب الإجمالي
- **خصم السلف**: يدوي حسب المبلغ المطلوب

### **4. العملة الافتراضية: الجنيه المصري**
- **جميع المبالغ** تظهر بالجنيه المصري (ج.م)
- **تنسيق احترافي** للأرقام
- **رمز العملة** واضح في كل مكان

## 🔢 **الحسابات المتقدمة**

### **الراتب النسبي حسب أيام العمل:**
```javascript
// مثال: راتب أساسي 6600 ج.م، غياب 3 أيام
const basicSalary = 6600        // ج.م
const workingDaysInMonth = 22   // يوم
const absentDays = 3            // يوم

const dailySalary = 6600 / 22 = 300 ج.م/يوم
const actualWorkingDays = 22 - 3 = 19 يوم
const proportionalSalary = 300 × 19 = 5700 ج.م
```

### **الخصومات التلقائية:**
```javascript
// خصم الغياب
const absentDeduction = 300 × 3 = 900 ج.م

// خصم التأخير (يومين تأخير)
const lateDeduction = 2 × (300 × 0.1) = 60 ج.م

// التأمينات الاجتماعية (9%)
const grossSalary = 5700 + 500 = 6200 ج.م (مع البدلات)
const insuranceDeduction = 6200 × 0.09 = 558 ج.م

// خصم السلف (حسب المطلوب)
const loanDeduction = 500 ج.م // مثال
```

### **الراتب الصافي النهائي:**
```javascript
const netSalary = grossSalary - totalDeductions
const netSalary = 6200 - (900 + 60 + 558 + 500) = 4182 ج.م
```

## 🔧 **الميزات المتقدمة**

- **تفاصيل الحسابات** خطوة بخطوة
- **شرح كل عملية** حسابية
- **إظهار/إخفاء** التفاصيل

## 🚀 **الفوائد العملية**

### **للمحاسبين:**
1. **حسابات دقيقة** حسب أيام العمل الفعلية
2. **خصم السلف** بسهولة ووضوح
3. **تقارير شاملة** لكل موظف
4. **عملة مألوفة** (الجنيه المصري)

### **للموارد البشرية:**
1. **متابعة الحضور** والغياب بدقة
2. **حساب العقوبات** تلقائياً
3. **إدارة السلف** والاستقطاعات
4. **تقييم الأداء** بالنسب

### **للإدارة:**
1. **شفافية كاملة** في الحسابات
2. **تقارير مفصلة** لكل موظف
3. **توفير الوقت** في المراجعة
4. **دقة عالية** في النتائج

## 📞 **الدعم والمساعدة**

### **للحصول على المساعدة:**
1. **راجع التبويبات** بالترتيب
2. **استخدم الحساب التلقائي** للدقة
3. **اعرض التفاصيل** لفهم الحسابات
4. **تواصل مع الدعم التقني** عند الحاجة

### **نصائح للاستخدام الأمثل:**
- **ابدأ بالبيانات الأساسية** ثم انتقل للتبويبات
- **تأكد من صحة أيام الغياب** قبل الحفظ
- **راجع الملخص** قبل الحفظ النهائي
- **استخدم عرض التفاصيل** للتحقق من الحسابات
