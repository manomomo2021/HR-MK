# نظام إدارة الموظفين المصري المحدث

## نظرة عامة

تم تحديث نظام إدارة الموظفين ليكون مناسباً للبيئة المصرية مع إضافة حقل كود الموظف وتحويل العملة إلى الجنيه المصري.

## 🇪🇬 **التحديثات المصرية**

### **1. حقل كود الموظف:**
- ✅ **إضافة حقل كود الموظف** في النموذج
- ✅ **عرض الكود في الجدول** كأول عمود
- ✅ **البحث بالكود** في خانة البحث
- ✅ **تصدير الكود** في ملفات Excel

### **2. الرقم القومي المصري:**
- ✅ **14 رقم** بدلاً من 10 أرقام
- ✅ **التحقق من صحة الرقم** القومي المصري
- ✅ **التحقق من تاريخ الميلاد** في الرقم القومي
- ✅ **دعم القرن العشرين والواحد والعشرين**

### **3. أرقام الهاتف المصرية:**
- ✅ **الهاتف المحمول**: 01xxxxxxxxx
- ✅ **مع كود الدولة**: +201xxxxxxxxx أو 00201xxxxxxxxx
- ✅ **الهاتف الأرضي**: 0xxxxxxx أو 0xxxxxxxx
- ✅ **دعم جميع شبكات المحمول** المصرية

### **4. العملة المصرية:**
- ✅ **الجنيه المصري (ج.م)** كعملة أساسية
- ✅ **رمز العملة** في جميع حقول الراتب
- ✅ **تنسيق مصري** للأرقام والعملة

## 🔍 **التحقق من صحة البيانات**

### **الرقم القومي المصري:**
```javascript
const validateEgyptianNationalId = (id) => {
  if (!id || id.length !== 14) return false
  
  // التحقق من أن جميع الأحرف أرقام
  if (!/^\d+$/.test(id)) return false
  
  // التحقق من صحة تاريخ الميلاد في الرقم القومي
  const century = id.charAt(0)
  if (!['2', '3'].includes(century)) return false // 2 للقرن العشرين، 3 للقرن الواحد والعشرين
  
  const year = parseInt(id.substring(1, 3))
  const month = parseInt(id.substring(3, 5))
  const day = parseInt(id.substring(5, 7))
  
  // التحقق من صحة الشهر واليوم
  if (month < 1 || month > 12) return false
  if (day < 1 || day > 31) return false
  
  return true
}
```

### **رقم الهاتف المصري:**
```javascript
const validateEgyptianPhone = (phone) => {
  if (!phone) return false
  
  // إزالة المسافات والرموز
  const cleanPhone = phone.replace(/[\s\-\(\)]/g, '')
  
  // التحقق من أرقام الهاتف المصرية
  const patterns = [
    /^01[0125]\d{8}$/, // الهاتف المحمول: 01xxxxxxxxx
    /^\+201[0125]\d{8}$/, // الهاتف المحمول مع كود الدولة: +201xxxxxxxxx
    /^00201[0125]\d{8}$/, // الهاتف المحمول مع كود الدولة: 00201xxxxxxxxx
    /^0[2-9]\d{6,7}$/, // الهاتف الأرضي: 0xxxxxxx أو 0xxxxxxxx
    /^\+20[2-9]\d{6,7}$/, // الهاتف الأرضي مع كود الدولة: +20xxxxxxx
    /^0020[2-9]\d{6,7}$/ // الهاتف الأرضي مع كود الدولة: 0020xxxxxxx
  ]
  
  return patterns.some(pattern => pattern.test(cleanPhone))
}
```

## 📞 **الاستخدام**

### **لإضافة موظف جديد:**
1. انقر على "إضافة موظف جديد"
2. أدخل كود الموظف (مثال: EMP001)
3. أدخل الاسم الكامل
4. أدخل الرقم القومي المصري (14 رقم)
5. أدخل رقم الهاتف المصري
6. أدخل الراتب بالجنيه المصري
7. احفظ البيانات

### **للبحث عن موظف:**
1. استخدم خانة البحث
2. ابحث بالكود أو الاسم أو الرقم القومي
3. استخدم فلاتر القسم ونوع العقد

### **للتصدير:**
1. انقر على "تصدير Excel"
2. احصل على ملف بالتنسيق المصري
3. استخدم الملف في البرامج المحاسبية
