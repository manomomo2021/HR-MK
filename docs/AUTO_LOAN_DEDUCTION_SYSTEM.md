# نظام خصم السلف التلقائي

## نظرة عامة

تم تطوير نظام متقدم لخصم السلف تلقائياً من رواتب الموظفين عند حساب الرواتب الشهرية، مع ضوابط أمان وحدود قانونية.

## 🎯 **الميزات الرئيسية**

### **1. الخصم التلقائي الذكي**
- ✅ **خصم تلقائي** عند حساب الرواتب
- ✅ **حدود آمنة** (25% كحد أقصى من الراتب)
- ✅ **تحديث فوري** لحالة السلف
- ✅ **إشعارات واضحة** للمحاسبين

### **2. ضوابط الأمان**
- 🛡️ **الحد الأقصى**: 25% من الراتب الأساسي
- 🛡️ **التحقق المسبق** قبل الموافقة على السلفة
- 🛡️ **منع الخصم الزائد** عن المبلغ المتبقي
- 🛡️ **إيقاف تلقائي** عند اكتمال السداد

## 💰 **كيفية عمل النظام**

### **1. طلب السلفة مع الخصم التلقائي**

#### **في شاشة طلب السلفة:**
```javascript
// حسابات الخصم التلقائي
const monthlyDeduction = loanAmount / installments
const maxAllowedDeduction = employeeSalary * 0.25  // 25% حد أقصى
const isDeductionValid = monthlyDeduction <= maxAllowedDeduction
```

#### **مثال عملي:**
```
راتب الموظف: 8,000.00 ج.م
مبلغ السلفة: 12,000.00 ج.م
عدد الأقساط: 6 أشهر

الحسابات:
📊 القسط الشهري: 12,000 ÷ 6 = 2,000.00 ج.م
📊 الحد الأقصى المسموح: 8,000 × 25% = 2,000.00 ج.م
✅ النتيجة: مقبول (ضمن الحدود المسموحة)
```

### **2. واجهة طلب السلفة المحسنة**

#### **إعدادات الخصم التلقائي:**
```

معلومات الموظف:
├── راتب الموظف الحالي: 8,000.00 ج.م
└── الحد الأقصى للخصم (25%): 2,000.00 ج.م

حسابات الخصم الشهري:
├── قيمة القسط الشهري: 2,000.00 ج.م
├── نسبة من الراتب: 25.0%

```

#### **التحذيرات الذكية:**
```

   يرجى تقليل مبلغ السلفة أو زيادة عدد الأقساط

✅ الخصم ضمن الحدود المسموحة
   سيتم خصم 2,000.00 ج.م شهرياً لمدة 6 شهر
```

### **3. حساب الرواتب مع الخصم التلقائي**

#### **عملية الخصم:**
```javascript
// البحث عن السلف المفعلة للخصم التلقائي
const employeeLoans = loans.filter(l => 
  l.employeeId === employee.id && 
  l.status === 'approved' && 
  l.autoDeduction === true &&
  (l.paidAmount || 0) < l.amount
)

// حساب الخصم لكل سلفة
for (const loan of employeeLoans) {
  const monthlyDeduction = loan.monthlyDeduction || loan.monthlyInstallment
  const remainingAmount = loan.amount - (loan.paidAmount || 0)
  const actualDeduction = Math.min(monthlyDeduction, remainingAmount)
  
  // تحديث حالة السلفة
  loan.paidAmount += actualDeduction
  if (loan.paidAmount >= loan.amount) {
    loan.status = 'completed'
  }
}
```

#### **مثال تطبيقي:**
```
الموظف: أحمد محمد
الراتب الأساسي: 8,000.00 ج.م
السلفة المستحقة: 2,000.00 ج.م (من أصل 12,000.00 ج.م)

حساب الراتب:
├── الراتب الإجمالي: 8,500.00 ج.م (مع البدلات)
├── خصم السلف: 2,000.00 ج.م ✅
├── خصومات أخرى: 765.00 ج.م (تأمينات)
└── الراتب الصافي: 5,735.00 ج.م

تحديث السلفة:
├── المبلغ المسدد: 4,000.00 ج.م (كان 2,000.00 ج.م)
├── المبلغ المتبقي: 8,000.00 ج.م
└── الحالة: موافق عليها (مستمرة)
```
