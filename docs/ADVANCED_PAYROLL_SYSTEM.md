# 💰 نظام حساب الرواتب المتقدم

## نظرة عامة

تم تطوير نظام متقدم وشامل لحساب الرواتب يطبق المعادلة المطلوبة: **الراتب الأساسي + البدلات = الإجمالي - خصم السلف - الخصومات = الصافي** مع حساب سعر اليوم الصافي ومعدل الحضور.

---

## 🎯 **المعادلة المطبقة**

### **المعادلة الأساسية:**
```
الراتب الأساسي + البدلات = الإجمالي
الإجمالي - خصم السلف - الخصومات = الصافي
```

### **حساب سعر اليوم:**
```
سعر اليوم الصافي = الراتب الصافي ÷ أيام العمل الفعلية
معدل الحضور = (أيام العمل ÷ أيام الشهر) × 100
```

---

## 🔧 **مكونات النظام**

### **1. الراتب الأساسي والبدلات:**

#### **الراتب الأساسي:**
- يؤخذ من بيانات الموظف (`basicSalary` أو `salary`)
- القيمة الافتراضية: 5000 ج.م

#### **البدلات المشمولة:**
- **بدل المواصلات** (`transportAllowance`)
- **بدل السكن** (`housingAllowance`) 
- **بدل الوجبات** (`mealAllowance`)
- **بدل الهاتف** (`phoneAllowance`)
- **بدل الوقت الإضافي** (`overtimeAllowance`)
- **بدلات أخرى** (`otherAllowances`)

#### **حساب بدل الوقت الإضافي:**
```javascript
const overtimeRate = parseFloat(employee.overtimeRate) || 1.5 // معدل الوقت الإضافي
const overtimeAllowance = totalOvertimeHours * hourlySalary * overtimeRate
```

### **2. الخصومات:**

#### **خصومات الحضور:**
- **خصم الغياب**: `أيام الغياب × سعر اليوم`
- **خصم التأخير**: `أيام التأخير × (سعر اليوم × 0.1)`
- **خصم التأخير الشديد**: `أيام التأخير > 60 دقيقة × (سعر اليوم × 0.5)`

#### **خصم السلف:**
- خصم تلقائي من السلف المعتمدة
- يحسب حسب القسط الشهري المحدد
- يتوقف عند سداد السلفة بالكامل

#### **الخصومات الإجبارية:**
- **التأمينات الاجتماعية**: `الراتب الأساسي × 0.09` (9%)
- **الضرائب**: `(الراتب الإجمالي - 15000) × 0.1` للرواتب > 15000

#### **خصومات أخرى:**
- **خصومات ثابتة** (`fixedDeductions`)
- **خصومات تأديبية** (`disciplinaryDeductions`)

### **3. حساب أيام العمل:**

#### **أيام الشهر الفعلية:**
```javascript
const [year, month] = selectedMonth.split('-')
const daysInMonth = new Date(year, month, 0).getDate()
const workingDaysInMonth = Math.floor(daysInMonth * 22 / 30) // تقدير أيام العمل
```

#### **أيام العمل الفعلية:**
- يحسب من سجلات الحضور الفعلية
- يشمل أيام الحضور والتأخير
- لا يشمل أيام الغياب

## 📊 **الحسابات المتقدمة**

### **1. سعر اليوم:**
```javascript
const dailySalary = basicSalary / workingDaysInMonth // سعر اليوم الأساسي
const dailyNetSalary = workingDays > 0 ? netSalary / workingDays : 0 // سعر اليوم الصافي
const effectiveDailySalary = workingDays > 0 ? grossSalary / workingDays : dailySalary // سعر اليوم الفعلي
```

### **2. سعر الساعة:**
```javascript
const hourlySalary = dailySalary / 8 // افتراض 8 ساعات عمل يومياً
```

### **3. معدل الحضور:**
```javascript
const attendanceRate = workingDays > 0 ? ((workingDays / workingDaysInMonth) * 100).toFixed(1) + '%' : '0%'
```

- 🎯 **معادلة واضحة ومرئية**: الراتب الأساسي + البدلات = الإجمالي - خصم السلف - الخصومات = الصافي

- 🎨 **واجهة جذابة ومفصلة** مع نافذة تفاصيل شاملة
