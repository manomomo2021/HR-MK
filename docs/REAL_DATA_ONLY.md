## 🔍 **أنواع الأخطاء المعالجة**

### **1. خطأ انتهاء المهلة (ETIMEDOUT):**
- **السبب**: الجهاز لا يستجيب خلال 5 ثوان
- **الحل**: التحقق من اتصال الشبكة

### **2. خطأ رفض الاتصال (ECONNREFUSED):**
- **السبب**: الجهاز غير متاح أو المنفذ مغلق
- **الحل**: التحقق من تشغيل الجهاز والمنفذ

### **3. خطأ المكتبة غير متوفرة:**
- **السبب**: مكتبة node-zklib غير مثبتة
- **الحل**: تشغيل `npm install node-zklib`

## 🛠️ **متطلبات التشغيل**

### **1. جهاز البصمة:**
- ✅ **متصل بالشبكة**: نفس شبكة الخادم
- ✅ **عنوان IP ثابت**: 192.168.0.201
- ✅ **المنفذ مفتوح**: 4370
- ✅ **الجهاز يعمل**: حالة تشغيل طبيعية

### **2. إعدادات الشبكة:**
```
IP Address: 192.168.0.201
Port: 4370
Protocol: TCP
Timeout: 5 seconds
```

### **3. المكتبات المطلوبة:**
```bash
npm install node-zklib
```

## 🔧 **استكشاف الأخطاء**

### **إذا ظهر خطأ ETIMEDOUT:**
1. تحقق من اتصال الشبكة: `ping 192.168.0.201`
2. تحقق من المنفذ: `telnet 192.168.0.201 4370`
3. أعد تشغيل جهاز البصمة
4. تحقق من إعدادات الشبكة في الجهاز

### **إذا ظهر خطأ ECONNREFUSED:**
1. تأكد من تشغيل جهاز البصمة
2. تحقق من إعدادات TCP/IP في الجهاز
3. تأكد من عدم وجود جدار حماية يحجب المنفذ
4. تحقق من عنوان IP الصحيح

### **إذا ظهر خطأ المكتبة:**
```bash
# تثبيت المكتبة
npm install node-zklib

# في حالة مشاكل الترجمة
npm install node-zklib --build-from-source

# أو استخدام yarn
yarn add node-zklib
```
