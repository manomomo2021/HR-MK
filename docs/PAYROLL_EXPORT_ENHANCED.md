- ⚡ **سرعة في التصدير**

- 🎯 **للاستخدام السريع**

```javascript
// حساب الإحصائيات التلقائية
const averageAttendance = data.payrolls.length > 0 ? 
  (totalWorkingDays / (data.payrolls.length * 22)) * 100 : 0

const totalLoanDeductions = data.payrolls.reduce((sum, p) => {
  const amount = parseFloat(p['خصم السلف'].replace(/[^\d.-]/g, '')) || 0
  return sum + amount
}, 0)
```

## 📞 **الاستخدام**

### **للتصدير الشامل:**
1. اختر الشهر المطلوب
2. اختر العملة للعرض
3. انقر على "تصدير تقرير شامل"
4. احفظ الملف واستخدمه للتحليل

### **للتصدير المبسط:**
1. اختر الشهر المطلوب
2. انقر على "تصدير مبسط"
3. احصل على ملف سريع للمراجعة

### **نصائح للاستخدام:**
- **استخدم التقرير الشامل** للتحليل المفصل
- **استخدم التقرير المبسط** للمراجعة السريعة
- **اختر العملة المناسبة** قبل التصدير
- **راجع الإحصائيات** في بداية التقرير
