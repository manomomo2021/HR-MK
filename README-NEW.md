# نظام إدارة الموارد البشرية - Next.js

نظام شامل لإدارة الموارد البشرية مطور بـ Next.js مع React وتصميم Neumorphism UI الأنيق.

- إضافة وتعديل وحذف بيانات الموظفين
- إدارة البيانات الشخصية والوظيفية
- تصدير بيانات الموظفين
- البحث والفلترة المتقدمة

### 🖥️ إدارة الأجهزة الثانوية
- إدارة أجهزة البصمة

- ربط الموظفين بالأجهزة

- إدارة أرصدة الأجازات السنوية
- تسجيل ومتابعة طلبات الأجازات
- أنواع مختلفة من الأجازات (سنوية، عارضة، مرضية)
- تحديث الأرصدة بشكل جماعي

### ⏰ الحضور والانصراف
- عرض سجلات الحضور والانصراف
- مزامنة البيانات من أجهزة البصمة
- إحصائيات الحضور اليومية
- تصدير تقارير الحضور

### 📊 التقارير

### ⚙️ الإعدادات

## التقنيات المستخدمة

- **Next.js 14** - إطار عمل React للتطبيقات الحديثة
- **React 18** - مكتبة واجهة المستخدم
- **Tailwind CSS** - إطار عمل CSS للتصميم السريع
- **Framer Motion** - مكتبة الحركة والتأثيرات
- **React Hook Form** - إدارة النماذج
- **React Hot Toast** - الإشعارات التفاعلية
- **Context API** - إدارة حالة التطبيق
- **Better SQLite3** - قاعدة بيانات SQLite محسّنة لـ Node.js
- **Font Awesome** - الأيقونات
- **Google Fonts (Cairo)** - الخطوط العربية

## متطلبات التشغيل

- Node.js 18+
- npm أو yarn
- متصفح ويب حديث يدعم ES6+

## التثبيت والتشغيل

1. **تحميل الملفات:**
   ```bash
   git clone [repository-url]
   cd hr-management-system
   ```

2. **تثبيت التبعيات:**
   ```bash
   npm install
   # أو
   yarn install
   ```

3. **تهيئة قاعدة البيانات:**
   ```bash
   # سيتم إنشاء قاعدة البيانات تلقائياً عند أول تشغيل للتطبيق
   # أو يمكنك تهيئتها يدوياً عبر استدعاء /api/init-db
   ```

4. **تشغيل التطبيق في وضع التطوير:**
   ```bash
   npm run dev
   # أو
   yarn dev
   ```

5. **فتح التطبيق:**
   - افتح المتصفح وانتقل إلى `http://localhost:3000`

6. **تسجيل الدخول:**
   - اسم المستخدم: `admin`
   - كلمة المرور: `admin123`

7. **بناء التطبيق للإنتاج:**
   ```bash
   npm run build
   npm run start
   # أو
   yarn build
   yarn start
   ```

## قاعدة البيانات

### هيكل قاعدة البيانات

تم تصميم قاعدة البيانات SQLite لتشمل الجداول التالية:

1. **المستخدمين (users)**: لتخزين معلومات المستخدمين الذين يمكنهم الوصول إلى النظام
2. **الأقسام (departments)**: لتخزين معلومات الأقسام المختلفة في الشركة
3. **المناصب (positions)**: لتخزين المناصب الوظيفية المختلفة
4. **الموظفين (employees)**: لتخزين معلومات الموظفين
5. **الحضور والانصراف (attendance)**: لتسجيل حضور وانصراف الموظفين
6. **الإجازات (leaves)**: لتسجيل طلبات الإجازات والموافقات عليها
7. **الرواتب (payroll)**: لتخزين معلومات الرواتب والمدفوعات
8. **أجهزة البصمة (biometric_devices)**: لتخزين معلومات أجهزة البصمة
9. **الإعدادات (settings)**: لتخزين إعدادات النظام المختلفة

### استخدام قاعدة البيانات

يتم التعامل مع قاعدة البيانات من خلال واجهات برمجة التطبيقات (APIs) المتاحة:

- `/api/init-db`: لتهيئة قاعدة البيانات وإنشاء الجداول
- `/api/employees`: للتعامل مع بيانات الموظفين
- `/api/attendance-sqlite`: للتعامل مع سجلات الحضور والانصراف
- `/api/payroll`: للتعامل مع بيانات الرواتب
- `/api/loans`: للتعامل مع بيانات السلف

## الاستخدام

### تسجيل الدخول
1. افتح النظام في المتصفح
2. أدخل بيانات تسجيل الدخول
3. انقر على "تسجيل الدخول"

### إدارة الموظفين
1. انتقل إلى صفحة "الموظفين"
2. انقر على "إضافة موظف جديد"
3. املأ البيانات المطلوبة
4. احفظ البيانات

### إدارة الأجازات
1. انتقل إلى "الأجازات السنوية" لعرض الأرصدة
2. استخدم "تسجيل الأجازات" لإضافة طلبات جديدة
3. يمكن تحديث الأرصدة بشكل جماعي

### إنشاء التقارير
1. انتقل إلى صفحة "التقارير"
2. اختر نوع التقرير المطلوب
3. حدد الفترة الزمنية والفلاتر
4. انقر على "إنشاء التقرير"
