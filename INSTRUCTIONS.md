# تعليمات تشغيل نظام إدارة الموارد البشرية - Next.js

## خطوات التشغيل

### 1. تثبيت Node.js
تأكد من تثبيت Node.js الإصدار 18 أو أحدث من الموقع الرسمي:
https://nodejs.org/

### 2. تثبيت التبعيات
افتح Terminal أو Command Prompt في مجلد المشروع وقم بتشغيل:

```bash
npm install
```

أو إذا كنت تستخدم yarn:

```bash
yarn install
```

### 3. تشغيل التطبيق
بعد تثبيت التبعيات، قم بتشغيل:

```bash
npm run dev
```

أو:

```bash
yarn dev
```

### 4. فتح التطبيق
افتح المتصفح وانتقل إلى:
```
http://localhost:3000
```

### 5. تسجيل الدخول
استخدم البيانات التالية لتسجيل الدخول:
- **اسم المستخدم:** admin
- **كلمة المرور:** admin123

## إصلاح المشاكل الشائعة

### مشكلة: خطأ في تثبيت التبعيات
**الحل:**
```bash
# مسح مجلد node_modules
rm -rf node_modules
# أو في Windows
rmdir /s node_modules

# مسح package-lock.json
rm package-lock.json
# أو في Windows
del package-lock.json

# إعادة تثبيت التبعيات
npm install
```

### مشكلة: خطأ في تشغيل التطبيق
**الحل:**
```bash
# التأكد من إصدار Node.js
node --version

# يجب أن يكون 18 أو أحدث
# إذا كان أقل، قم بتحديث Node.js
```

### مشكلة: البورت 3000 مستخدم
**الحل:**
```bash
# تشغيل على بورت مختلف
npm run dev -- -p 3001
```

### مشكلة: خطأ في Tailwind CSS
**الحل:**
```bash
# إعادة بناء Tailwind
npx tailwindcss -i ./styles/globals.css -o ./styles/output.css --watch
```

## بناء التطبيق للإنتاج

```bash
# بناء التطبيق
npm run build

# تشغيل النسخة المبنية
npm run start
```

## تصدير التطبيق كملفات ثابتة

```bash
# بناء وتصدير
npm run build
npm run export

# الملفات ستكون في مجلد out/
```
