# دليل التثبيت - نظام البصمة

## 📋 **المتطلبات الأساسية**

### **1. متطلبات النظام:**
- Node.js 16.0 أو أحدث
- npm أو yarn
- جهاز بصمة ZKTeco متصل بالشبكة

### **2. متطلبات الشبكة:**
- اتصال بالإنترنت لتحميل المكتبات
- اتصال شبكة محلية مع جهاز البصمة
- المنفذ 4370 مفتوح للاتصال بالجهاز

## 🚀 **خطوات التثبيت**

### **الخطوة 1: تثبيت المكتبات**
```bash
# تثبيت مكتبة جهاز البصمة
npm install node-zklib

# تثبيت مكتبات التصدير
npm install xlsx file-saver

# أو تثبيت جميع المكتبات مرة واحدة
npm install node-zklib xlsx file-saver
```

### **الخطوة 2: إعداد متغيرات البيئة**
أنشئ ملف `.env.local` في جذر المشروع:
```env
# إعدادات جهاز البصمة
BIOMETRIC_IP=192.168.0.201
BIOMETRIC_PORT=4370
BIOMETRIC_TIMEOUT=10000
BIOMETRIC_INPORT=4000

# إعدادات التطوير
NODE_ENV=development
```

### **الخطوة 3: إعداد بيانات الموظفين**
عدل ملف `data/employees.js` وأضف بيانات موظفيك:
```javascript
const employees = [
  {
    code: '101',                    // كود البصمة (مهم جداً)
    name: 'أحمد محمد السعيد',
    salary: 8000,
    department: 'المحاسبة',
    position: 'محاسب أول',
    joinDate: '2023-01-15'
  },
  // أضف المزيد من الموظفين...
]
```

### **الخطوة 4: إعداد جهاز البصمة**
تأكد من:
- ✅ الجهاز متصل بالشبكة
- ✅ عنوان IP صحيح (192.168.0.201)
- ✅ المنفذ 4370 مفتوح
- ✅ الجهاز يعمل بشكل طبيعي

### **الخطوة 5: اختبار الاتصال**
```bash
# تشغيل الخادم
npm run dev

# في متصفح آخر أو Postman
curl http://localhost:3000/api/attendance
```

## 🔧 **إعداد الجهاز**

### **1. إعدادات الشبكة:**
```
IP Address: 192.168.0.201
Subnet Mask: 255.255.255.0
Gateway: 192.168.0.1
Port: 4370
```

### **2. إعدادات الاتصال:**
- تأكد من تفعيل TCP/IP
- تأكد من عدم وجود كلمة مرور للجهاز
- تأكد من تفعيل Real-time events

### **3. إعدادات المستخدمين:**
- أضف الموظفين في الجهاز
- تأكد من تطابق الأكواد مع ملف employees.js
- سجل بصمات الموظفين

## 🧪 **اختبار النظام**

### **1. اختبار الاتصال:**
```javascript
// في وحدة تحكم المتصفح
fetch('/api/attendance')
  .then(res => res.json())
  .then(data => {
    if (data.success) {
      console.log('✅ الاتصال ناجح:', data.summary)
    } else {
      console.error('❌ فشل الاتصال:', data.message)
    }
  })
```

### **2. اختبار البيانات:**
```bash
# يجب أن ترى استجابة مثل:
{
  "success": true,
  "attendance": [...],
  "summary": {
    "totalRecords": 125,
    "totalEmployees": 20,
    "lastUpdate": "2024-01-15T10:30:25.000Z"
  }
}
```

### **3. اختبار الواجهة:**
انتقل إلى: `http://localhost:3000/biometric-attendance`

## 🚨 **حل المشاكل الشائعة**

### **مشكلة: "لا يمكن الاتصال بجهاز البصمة"**
```bash
# تحقق من الاتصال
ping 192.168.0.201

# تحقق من المنفذ
telnet 192.168.0.201 4370
```

**الحلول:**
1. تأكد من أن الجهاز متصل بالشبكة
2. تحقق من عنوان IP في إعدادات الجهاز
3. تأكد من عدم وجود جدار حماية يحجب الاتصال
4. أعد تشغيل الجهاز

### **مشكلة: "البيانات فارغة"**
**الحلول:**
1. تأكد من وجود سجلات حضور في الجهاز
2. تحقق من تطابق أكواد الموظفين
3. تأكد من صحة تاريخ ووقت الجهاز

### **مشكلة: "خطأ في تثبيت node-zklib"**
```bash
# للأنظمة التي تتطلب Python
npm install --python=python2.7

# أو استخدم yarn
yarn add node-zklib

# في حالة مشاكل الترجمة
npm install --build-from-source
```

### **مشكلة: "انتهاء مهلة الاتصال"**
عدل إعدادات المهلة في `.env.local`:
```env
BIOMETRIC_TIMEOUT=20000  # 20 ثانية بدلاً من 10
```
